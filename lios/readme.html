Linux-Intelligent-OCR-Solution (Lios)

What is Lios?

Lios is a free and open source software for converting print into text using either a scanner or a camera. It can also produce text out of scanned images from other sources such as PDF, Image, Folder containing Images or screenshot. The program is given total accessibility for visually impaired. Lios is written in Python 3 and released under the GPL-3 license.

Features of Lios

  • Import images from Scanner, PDFs, Folder, or Webcam
  • Take and Recognize Screenshot
  • Recognize Selected Areas (Rectangle selection)
  • Support two OCR Engines (Cuneiform, Tesseract)
  • 24 Language support (30 more languages can be installed in Tesseract)
  • Full Auto Rotation for any Language
  • Side by side view of image and output
  • Advanced Scanner Brightness optimizer
  • Text Reader for low vision with Highlighting
  • Audio converter (espeak)
  • Spell-checker (aspell)
  • Export as PDF (text/images)
  • Dictionary Support for English (Artha)
  • Options for save, load and reset settings

Opening and Closing

To open Lios, launch Application menu > Graphics > Lios. Or search in the dashboard and start Lios from there. To close, select Quit from File menu or press Ctrl+Q.

Importing Images

Lios supports importing images from various sources, including:

  • Single or multiple image files
  • PDF files
  • Folders containing images
  • Scanner
  • Screenshot
  • Webcam

OCR and Text Manipulation

Lios offers various OCR options and text manipulation tools, including:

  • OCR with and without rotation
  • Rectangle selection for OCR
  • Text editing tools (cut, copy, paste, delete)
  • Find and Replace
  • Spell checker
  • Text reader with TTS

Utilities

Lios includes several useful utilities:

  • Brightness Optimizer for scanner
  • Audio converter
  • Artha dictionary integration
